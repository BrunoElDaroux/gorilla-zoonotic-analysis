TRANSPARENCY STATEMENT
-----------------------
No single public dataset exists that combines tourist respiratory screening
records with gorilla health event logs at the individual level. The datasets in
data/raw/ are SYNTHETIC, generated by notebooks/01_setup_and_data_generation.ipynb.

Every parameter used in the data generator (illness rates, seasonal patterns,
age-specific susceptibility, incubation windows, group sizes) is calibrated to
published peer-reviewed literature, cited in full below.

This is a standard approach in conservation epidemiology for:
  - Modeling health protocol efficacy before deployment
  - Training epidemiological analysis pipelines on representative data
  - Educational and portfolio purposes

If you gain access to real field data (e.g. through the Dian Fossey Gorilla
Fund, Gorilla Doctors, or Uganda Wildlife Authority), the pipeline in this
project is designed to accept that data with minimal modification.


ALTERNATIVE REAL DATASETS (for augmentation or replacement)
------------------------------------------------------------
The following public datasets can supplement or partially replace synthetic data:

1. GBIF — Mountain Gorilla Occurrence Records
   URL: https://www.gbif.org/species/5219404
   Access: Free, requires GBIF account for bulk download
   Format: Darwin Core Archive (CSV)
   Contains: Occurrence records, coordinates, dates
   Limitation: No health data

2. IUCN Red List — Gorilla beringei Population Data
   URL: https://www.iucnredlist.org/species/39994/115576640
   Access: Free
   Contains: Population estimates, threats, habitat data

3. Africa Biodiversity Collaborative Group (ABCG) Health Datasets
   URL: https://www.abcg.org/
   Access: Partial public access

4. Rwanda Development Board — Tourism Statistics
   URL: https://www.rdb.rw/tourism-statistics
   Contains: Annual tourist numbers, gorilla trekking permits
   Access: Annual reports (PDF)

5. Published Supplementary Data — Grützmacher et al. (2018)
   Journal: Scientific Reports
   DOI: https://doi.org/10.1038/s41598-018-25820-8
   Contains: Respiratory virus detection rates in gorillas and humans
   Access: Open access


DATASETS GENERATED (data/raw/)
--------------------------------

1. tourist_health_records.csv
   Rows: ~4,800 records (spanning 2015–2023, ~600/year)
   Columns:
     - tourist_id          : Unique tourist identifier (T-XXXXX)
     - visit_date          : Date of gorilla trek (YYYY-MM-DD)
     - group_visited       : Gorilla group name visited
     - had_symptoms        : Binary — reported respiratory symptoms at screening
     - test_result         : positive / negative / not_tested
     - symptom_severity    : mild / moderate / severe / none
     - age_group           : tourist age bracket (18-30, 31-45, 46-60, 61+)
     - nationality_region  : region of origin (Africa, Europe, N_America, etc.)
     - visit_duration_min  : Actual time spent with gorillas (capped at 60 min)
     - mask_compliant      : Whether tourist wore mask throughout visit
     - pre_screened        : Whether tourist completed pre-visit health screening
   Literature basis:
     - Positive respiratory symptom rate ~12-18% (Grützmacher et al. 2018)
     - Masking compliance variation by year (pre/post 2020 COVID policy shift)

2. gorilla_health_events.csv
   Rows: ~650 events (spanning 2015–2023)
   Columns:
     - event_id            : Unique event identifier (GHE-XXXXX)
     - gorilla_id          : Individual gorilla identifier (G-XXXX)
     - event_date          : Date illness was first observed
     - group_id            : Gorilla group
     - illness_type        : respiratory / gastrointestinal / dermatological / injury / other
     - severity            : mild / moderate / severe / fatal
     - days_ill            : Duration of illness episode (days)
     - recovered           : True/False
     - veterinary_treated  : True/False
     - notes               : Free text notes on transmission context
   Literature basis:
     - ~70-80% of great ape illness events are respiratory (Köndgen et al. 2008)
     - Fatality rate from respiratory illness: ~5-8% (Palacios et al. 2011)

3. gorilla_demographics.csv
   Rows: ~185 individual gorillas
   Columns:
     - gorilla_id          : Unique identifier (G-XXXX)
     - name                : Individual name (field convention)
     - group_id            : Social group affiliation
     - sex                 : M / F
     - age_years           : Estimated age
     - age_category        : infant / juvenile / subadult / adult_female /
                             adult_male / silverback
     - birth_year          : Estimated birth year
     - mother_id           : Mother's gorilla_id (if known)
     - status              : alive / deceased / unknown
     - immunocompromised   : Binary flag for known health vulnerabilities
   Literature basis:
     - Age category definitions: Robbins et al. (2004)
     - Group sizes: Virunga census data (Hickey et al. 2019)

4. tourist_visit_logs.csv
   Rows: ~4,800 (one per tourist per visit day)
   Columns:
     - visit_id            : Unique visit log entry
     - tourist_id          : Links to tourist_health_records.csv
     - visit_date          : Date of visit
     - group_id            : Gorilla group visited
     - duration_minutes    : Time spent within 7m of gorillas
     - min_distance_m      : Minimum recorded distance to gorillas (meters)
     - guide_id            : Guide identifier
     - group_size_tourists : Number of tourists in that trekking party
     - weather_condition   : clear / cloudy / light_rain / heavy_rain
     - season              : dry_long / dry_short / wet_long / wet_short
   Literature basis:
     - 8 tourist max per gorilla group per day (RDB regulations)
     - 7-meter minimum distance rule (DFGFI/IGCP protocol)

5. seasonal_climate_data.csv
   Rows: ~3,287 daily records (2015–2023)
   Columns:
     - date                : Calendar date
     - month               : Month number (1-12)
     - year                : Year
     - season              : Rwanda seasonal classification
     - rainfall_mm         : Daily rainfall (mm)
     - temp_max_c          : Maximum temperature (°C)
     - temp_min_c          : Minimum temperature (°C)
     - humidity_pct        : Relative humidity (%)
     - tourist_count       : Total tourists across all groups that day
     - gorilla_illness_count: Gorilla illness events recorded that day
   Source basis:
     - Rwanda climate normals from Rwanda Meteorological Agency
     - Virunga altitude: 2,200–4,507m ASL
     - Rwanda seasons: Mar-May (long wet), Jun-Sep (long dry),
                       Oct-Nov (short wet), Dec-Feb (short dry)
